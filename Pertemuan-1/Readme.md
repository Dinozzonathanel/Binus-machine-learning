# Laporan Analisis Data Iris - Dinozzo Nathanael Budiman

## Domain Proyek

Melakukan sebuah analisis tentang data Iris untuk mengetahui jenis data yang diperlukan sebelum belajar keseluruhan metode Machine Learning.

### Latar Belakang

Sebagai awal dari proses belajar/pelaksanaan saya (Sebagai tugas, bernama "Pertemuan-1") sebelum memulai melakukan proyek-proyek Machine Learning.

## Business Understanding

Analisis sederhana yang digunakan untuk memahami data yang telah digunakan, seperti melihat banyak baris dan kolomnya, fitur-fitur apa saja yang ada, dan apa yang dapat diprediksi dari proyek tersebut.

### Problem Statements

Bagaimana cara untuk membuat sebuah analisis sederhana tentang/pada data Iris.

### Goals

Proyek ini dibuat dan dilaksanakan untuk melihat fungsi dan guna dari data Iris serta dengan data yang telah digunakan.

### Solution statements

Dengan cara menggunakan program yang telah diberikan melalui websitenya kak Azhar saya dapat menggunakan itu untuk dimasukan ke dalam google collab untuk mengolah data dan melihat dari hasil yang diberikan, akhirnya saya dapat melihat data yang digunakan untuk membuat laporan ini dll.
