### Problem Statements

## Goals

# Pertanyaan penting
* Mengapa encoding diperlukan?
* Mengapa scaling diperlukan?
* Apa itu data leakage?

1. Encoding diperlukan karena machine learning hanya dapat menemerima angka dan tidak dapat menerima data dalam bentuk string
2. Scalling diperlukan agar machine tersebut tidak memprioritaskan nomor yang paling besar
3. Data leakage maksudnya info dari sebuah data test masuk ke dalam proses training model
